My Bird List app - Laravel and React

A work in progress... Since a very young age, I have been an avid birdwatcher and I kept track of the separate species I've seen in the back of a birdwatching guide I had. My sister 'borrowed' that a while ago so it's anyone's guess as to where it is now.... So, I decided to use my coding skills to write an app to keep track of my life list. My Bird List is an app that allows birdwatchers to store a list of the birds they've seen along with an image and location of the sighting. Built with a Laravel RESTful API back-end and using Laravel Sanctum for user auth, the frontend is written in React.js. Users can register and once logged in can add and edit their bird sightings (location set by clicking on an interactive map), Sightings are shown as a list and also via markers on a map.
